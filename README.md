# OpenDevEd header snapper

Use: 
* 3 mm sheet stock (~1/4"), e.g., clear acryllic.

Parts list:
* M6 x 16 mm hex bolt x4
* M6 nut x4 
* M3 x 16 mm machine screw x2
* M3 nut x2
